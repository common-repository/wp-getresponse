== Description ==

* Currently in Beta - Use at your own risk *

Uses the GetResponse JSON API ( no iFrames ) to add contacts to your specific campaigns automatically from your site. 

Available options to send new contacts:
* Add new registrations to GetResponse service
** Full name option available for registration page
* Add commenter's name and email after a comment ( planned feature )
* Customizable widget to be placed anywhere on your site for easy campaign signups ( planned feature )
* Post/Page short code available to insert a *form* into any post/page ( planned feature )

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the `wp-getresponse` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the options page found under Options > GetResponse.

== Frequently Asked Questions ==

= Can I pick which campaign to use? =

Yes, after you enter your API all your campaigns will show up in the options list.

= I don't see any campaigns in the campaign options =

Enter your API key and save.
